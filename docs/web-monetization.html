Web Monetization streaming

This page is Web-Monetized. If you have the Web Monetization browser extension installed, a tiny Interledger micropayment stream flows from you to the wallet pinned in the <link rel="monetization"> tag in the <head>.

Supported wallet providers (2026)

  • Chimoney — global, easiest
  • Fynbos — Canada only
  • GateHub — EU
  • Interledger Test Wallet — used by this page until production wallet is provisioned

What the money goes toward

  • skill-audit-mcp — MCP supply-chain auditor
  • secrets-audit-mcp — 32-provider secret scanner

Other ways to support

  • Polar.sh recurring tiers — $5 / $20 / $50
  • Drips Network — Ethereum continuous-rate funding
  • GitHub Sponsors
  • Bankr x402 endpoint — pay-per-scan on Base
